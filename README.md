## 🚀 Sobre o Projeto

O **Perfect Ads** é um projeto de portfólio desenvolvido para demonstrar a criação de uma experiência digital premium aliada a práticas modernas de **DevOps e entrega contínua**.

O projeto combina **design focado em conversão** com uma estrutura técnica simples, eficiente e totalmente automatizada.

---

## 🎯 Objetivo

Este projeto tem dois focos principais:

### 🎬 Experiência Digital

* Interface visual premium (Black & Gold)
* Foco em impacto e percepção de valor
* Estrutura otimizada para conversão

### ⚙️ DevOps / Engenharia

* Demonstrar pipeline CI/CD na prática
* Automatizar deploy
* Aplicar boas práticas com Git

---

## ⚙️ Pipeline (CI/CD)

```bash
Developer (Local)
        ↓
Git Commit
        ↓
GitHub (Repository)
        ↓
Vercel (Auto Deploy)
        ↓
Production (Live)
```

### 🔁 Fluxo de Deploy

* Alterações no código
* Commit + Push para o GitHub
* Deploy automático via Vercel
* Atualização instantânea em produção

> ⚡ Entrega contínua
> ⚡ Sem deploy manual
> ⚡ Pipeline simples e eficiente
